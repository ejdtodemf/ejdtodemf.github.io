## 개요
- 배열의 정의를 알 수 있습니다..

#### 배열이란?
- 배열 레퍼런스 변수 선언
- int[] x;
	- int x[];
	- x = new int[4];
- 배열 생성 및 배열 레퍼런스 변수에 대입
- 배열 변수 선언과 동시에 생성도 가능
	- int []x = new int[10];
- 배열 레퍼런스 변수 선언과 동시에 초기화 : 아래 두 가지 타입은 같은 의미
	-  int [] x = {1,2,3,4}
	- int [] x = new int[4];
	- x[0] = 1
	- x[1] = 2
	- x[2] = 3
	- x[3] = 4
